Projects

Crwn Clothes

My most up to date project, Crwn Clothing is a E-Commerce site built with React. This is by far the biggest project I've worked on and I've still got one or two things to add. So far the project is more Frontend oriented, using Firebase for the backend part. It's authentication capabilities are based on Firebases authentication service but it also uses firestore, the noSQL online database for item storage, as for the payments, they are handled by Stripe. This project helped me grow a lot when it comes to react and all the technologies that go with it. I developed a deeper understanding of redux, react-router, data flow and state management. As for the asynchronous actions, the live version of the website uses redux-thunk. Speaking of which, the newer version from github refactored some parts into Styled-Components , and Redux-Saga. In the near future I plan to use React Hooks and even GraphQL.

Smart Brain

Smart-Brain is a minimalist website with a login/register function which allows you to submit URLs of images, on which it uses the face detection model of the Clarifai API. It keeps track of the number of submits a user has made. For the backend I used Express.js and for the databases I chose PostgreSQL together with Knex.js for ease of use. Frontend still consists mainly of React and I plan to re-implement parts of it with Redux as an exercise. The website is currently deployed on Heroku.
